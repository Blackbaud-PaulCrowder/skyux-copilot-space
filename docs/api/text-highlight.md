#### SkyTextHighlightDirective

Highlights all matching text within the current DOM element.

**Selector:** `[skyHighlight]`

**Package:** `@skyux/indicators`

**Inputs:**

- `skyHighlight`: `string | string[] | undefined` - The text to highlight.
